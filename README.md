# Rinu's Portfolio

A clean, accessible, and anime‑inspired single‑page portfolio template built with HTML and CSS.

## Overview
This repository contains a portfolio landing page showcasing skills, projects, and contact information. It is designed to be:

- **Minimal**: Pure HTML + CSS, no build tools required
- **Accessible**: Semantic structure, ARIA labels, and form fields
- **Responsive**: Works on mobile and desktop
- **Stylish**: Clean layout with a subtle anime aesthetic

## What’s Included
- `index.html` — Main portfolio page with the following sections:
  - Hero
  - About
  - Skills
  - Projects
  - Timeline
  - Contact
- `style.css` — Styling for layout, typography, and components

## ▶Running Locally
1. Clone this repo:
   ```bash
   git clone https://github.com/Rinu-1705/portfolio.git
   cd portfolio
   ```
2. Open `index.html` in your browser.

> Tip: For best results, open the page through a local web server (e.g., `python -m http.server`) to avoid file protocol restrictions.

## Customization
- Replace the placeholder profile image path (currently a local absolute path) with a relative path or URL.
- Update the project cards and contact links to match your real work and profiles.

## Notes
- The contact form is static (no backend). You can connect it to a service like Formspree, Netlify Forms, or a server endpoint.
